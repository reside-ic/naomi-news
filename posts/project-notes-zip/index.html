Project notes added to output zip

<span title='2022-06-20 00:00:00 +0000 UTC'>20 Jun 2022</span>&nbsp;·&nbsp;<span>hint 2.2.0</span>

Project and version notes are now included in the model output zip. Now, when you write notes on the project page these will added to the output zip in a file “notes.txt”. The notes file contains the name, last modified time and content of the the project and every version.
